# 8ï¸âƒ£ Add Actions (increment / decrement)

  count: 0,

  increment: () =>
    set((state) => ({
      count: state.count + 1,
    })),

  decrement: () =>
    set((state) => ({
      count: state.count - 1,
    })),
}));

- set() accepts a function
- Function receives previous state
- Must return new object
